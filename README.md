## Project Overview
This project studies hallucination behavior in medical reasoning tasks.

We compare two open-weight large language models:

- Qwen
- Llama

We test them under two prompting settings:

- **Baseline prompt**
- **Cautious prompt**

We evaluate model behavior on four medical reasoning subsets:

- **FCT**: misleading context with wrong student answer
- **Fake**: false or absurd medical concepts
- **NOTA**: none-of-the-above reasoning
- **UET**: insufficient-evidence questions that require abstention

The goal is to understand:

- whether models are easily misled
- whether they guess when information is missing
- whether cautious prompting can reduce unsafe behavior

## Repository Structure

### `data/`
This folder contains the datasets used in the project.

Main files include:
- sampled_reasoning_FCT.csv
- sampled_reasoning_fake.csv
- sampled_reasoning_nota.csv
- uet_full_50_for_experiment.csv

### `outputs/`
This folder contains the raw generation outputs from the models.

Main files include:
- full_baseline_outputs（qwen）.csv
- full_cautious_outputs（qwen）.csv
- full_baseline_outputs（llama）.csv
- full_cautious_outputs（llama）.csv

These files are generated by the inference notebook.

### `eval_results/`
This folder contains processed evaluation results and figures.

It may include:
- summary tables
- metric results
- plots for FCT / Fake / NOTA / UET

### `train(qwen+Llama) (4).ipynb`
This is the main inference notebook for **Role 2**.

It:
- loads the datasets
- builds prompts
- runs Qwen and Llama
- saves baseline and cautious outputs

Note:
This notebook does **not** train or fine-tune the models.
It performs **zero-shot inference** only.

### `eval (1).ipynb`
This is the main evaluation notebook for **Role 3**.

It:
- loads the output CSV files
- computes evaluation metrics
- compares Qwen and Llama
- generates summary tables and figures

## Main Experiment Design

### Models
- Qwen
- Llama

### Prompt Settings
- **Baseline**: standard answering prompt
- **Cautious**: prompt with stronger safety and abstention instructions

### Evaluation Subsets
- **FCT**
  - tests whether the model is misled by wrong student reasoning
- **Fake**
  - tests whether the model can reject absurd medical concepts
- **NOTA**
  - tests whether the model can choose the exclusionary correct answer
- **UET**
  - tests whether the model can abstain when key evidence is missing
